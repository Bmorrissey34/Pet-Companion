# Benchmarks

## Overview

All benchmark scripts are located in the `bench/` directory and represent staged development of the robot navigation system.

## benchmark_simple.py

Initial proof-of-concept navigation script (original AI-generated version).

## benchmark_v2.py

Introduced camera head scanning and multiple observations.

## benchmark_v3.py

Added structured JSON outputs and logging.

## benchmark_v4.py

Hybrid navigation combining LLM reasoning with onboard sensors.

## benchmark_rewrite.py

Rewrite of the original `benchmark_simple.py` with updated commands to match actual PiCar function calls.
